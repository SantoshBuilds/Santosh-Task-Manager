# 📌 Team Task Manager

Team Task Manager is a full-stack MERN task management application built for teams and individuals to create, assign, track, and manage tasks in one place. It includes role-based access for admins and members, task progress tracking, checklist updates, team management, report downloads, and a responsive React interface.

## 🚀 Project Overview

This application helps teams organize daily work by allowing admins to create tasks, assign them to team members, monitor progress, and export reports. Members can view their assigned work, update checklist progress, open task details, and access task-related attachments or links.

## 🛠️ Tech Stack

- **Frontend:** React.js, Vite, Tailwind CSS
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **Authentication:** JWT

## 👑 Admin Panel

Admins can:

- View dashboard statistics
- Create tasks
- Assign tasks to users
- Manage all tasks
- Track task status and priority
- View team members
- Download task reports

## 👤 User Panel

Users can:

- View assigned tasks
- Track pending, in-progress, and completed tasks
- Update checklist progress
- View task details
- Access attachments or task links

## ✨ Key Features

- Role-based authentication with JWT
- Admin and member dashboards
- Task creation, assignment, update, and deletion
- Checklist-based task progress tracking
- Automatic task status updates based on checklist completion
- Priority and status indicators
- Team member management
- Task and user report downloads
- Attachment/link support for tasks
- Responsive UI for desktop, tablet, and mobile

## 🔐 Roles

Admin and member roles are stored in MongoDB using the `role` field:

```js
role: "admin"
role: "member"
```

- **Admin:** Can manage tasks, team members, dashboard data, and reports.
- **Member:** Can view assigned tasks and update task checklist progress.

## ⚙️ Environment Variables

Create a `.env` file inside the `backend/` folder:

```env

MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_jwt_secret
ADMIN_INVITE_TOKEN=your_admin_token
```

## 🧩 Setup Instructions

### 1. Clone the Repository

```bash
git clone https://github.com/SantoshBuilds/Santosh-Task-Manager.git
cd Santosh-Task-Manager
```

### 2. Install and Run Backend

```bash
cd backend
npm install
npm run dev
```

The backend runs on:

```text
http://localhost:5000
```

### 3. Install and Run Frontend

Open a new terminal:

```bash
cd frontend/Task-Manager
npm install
npm run dev
```

The frontend usually runs on:

```text
http://localhost:5173
```

If port `5173` is already in use, Vite may start on another port such as `5174`.
